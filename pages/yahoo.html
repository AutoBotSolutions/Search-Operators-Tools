SEARCH OPERATORS

All Words

"words" - All words must appear

Example: "Python programming"

Any Words

OR - Any word can appear

Example: tutorial OR guide

Exact Phrase

"phrase" - Exact phrase match

Example: "machine learning"

Exclusion

-term - Exclude term

Example: -advertisement

Site Filter

site: - Restrict to domain

Example: site:wikipedia.org

File Type

filetype: - Filter by type

Example: filetype:pdf

Language

lang: - Filter by language

Example: lang:english

Date Filter

date: - Filter by date

Example: date:past_week
